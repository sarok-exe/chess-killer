# Chess Killer

Chrome extension for chess analysis with Stockfish on Chess.com

## Install

1. Open Chrome and go to `chrome://extensions/`
2. Enable "Developer mode"
3. Click "Load unpacked"
4. Select the `chess-killer` folder

## Features

- Analyzes chess positions on Chess.com
- Shows best move with Stockfish
- Clean overlay panel
- For analysis only - don't use in rated games!

## Files

- `manifest.json` - Extension config
- `content.js` - Runs on Chess.com
- `popup.html/js` - Extension popup UI
